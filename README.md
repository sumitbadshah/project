Smart Attendance System
A Python-based face recognition attendance system that uses computer vision and a simple GUI to track and manage attendance automatically.

Features
Real-time Face Detection: Uses OpenCV's Haar Cascade for face detection
Face Recognition: Custom simple face recognizer using similarity comparison
Attendance Tracking: Automatically marks attendance when recognized faces appear
Database Storage: SQLite database for storing face data and attendance records
User-friendly GUI: PyQt5-based interface with live camera feed
Face Registration: Easy registration of new faces with name and ID
Attendance Records: View and manage attendance history

Requirements
Python Packages
Python 3.6+
OpenCV (opencv-python)
NumPy
PyQt5
SQLite3 (built-in)

Hardware
Webcam
